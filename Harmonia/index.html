Comprehensive Features

Everything needed to maintain independence while ensuring safety and peace of mind for families.

Smart Reminders

Gentle medication alerts and task notifications with voice input support to maintain routines.

Family Profiles

Photo-based recognition helps patients remember loved ones with visual cues.

Digital Journal

Record moods, thoughts, and memories through photos, notes, and voice logs.

Memory Games

Engaging exercises personalized with familiar faces to support cognition.

Accessibility Suite

Large buttons, voice controls, high contrast UI — easy for everyone to use.

Secure Backup

Optional Google Drive sync for data recovery — completely private and encrypted.

Emergency SOS

Instantly call a trusted contact using the SOS button on the home screen.

Offline Functionality

Works without internet — all data is saved locally for safety and privacy.

Geofence Alerts

Notifies caregivers if the patient moves over 100 meters from home.

Emotional Journal

Track emotional well-being through a daily mood log interface.

PIN Protection

Only caregivers with a PIN can edit personal or medical details.

Light & Dark Mode

User-friendly interface that adapts to different lighting preferences.

Technical Excellence

Built with Flutter and privacy-first architecture for reliable, secure, and accessible healthcare technology.

Built with Flutter

  • Cross-platform framework (Dart)
  • Material Design 3 UI components
  • Target: Android (APK)
  • Tunisia timezone support
  • Enhanced notification services

Privacy by Design

  • 100% offline operation
  • CSV-based local storage
  • File system storage for media
  • No cloud database dependency
  • Zero remote data transmission

Data Architecture

  • CSV files for structured data
  • Local file storage for images/audio
  • SharedPreferences for app settings
  • Complete data portability
  • No vendor lock-in

Key Dependencies

  • shared_preferences - App settings
  • path_provider - File system access
  • csv - Data parsing
  • url_launcher - Email integration
  • timezone - Tunisia support

Accessibility Focus

Background Services

Quick Installation Guide

Get Harmonia running on your Android device in just a few simple steps.

1

Download APK

Download the latest Harmonia APK file from our secure download link or scan the QR code.

# Download size: ~50MB # Compatible: Android 6.0+ # No Google Play Store required
2

Enable Unknown Sources

Allow installation from unknown sources in your Android settings if prompted.

Settings → Security → Install unknown apps OR Settings → Apps → Special access → Install unknown apps
3

Install Application

Open your Downloads folder, tap the APK file, and follow the installation prompts.

# Installation process: 1. Tap Harmonia.apk file 2. Tap "Install" button 3. Wait for installation to complete 4. Tap "Open" to launch
4

Setup & Configuration

Complete the initial setup process and configure notifications for optimal experience.

# First launch setup: - Profile creation - Emergency contacts - Notification permissions - Accessibility preferences
