Track Your Finances, Achieve Your Goals

Spendora is a personal finance tracking application that helps you manage your income, expenses, and savings goals with an intuitive and visual interface.

Python Backend Flutter App MySQL Data Visualization
Download Now

Core Features

User Authentication

Secure registration, login, and profile management to keep your financial data safe.

Income & Expense Management

Add, edit, delete, and categorize your financial transactions with ease.

Dashboard

Visualize your financial situation with interactive graphs and charts.

Financial Goals

Set savings goals and track your progress towards achieving them.

Reports & Insights

Generate monthly and annual reports to understand your spending habits.

CSV Export

Export your financial data in CSV format for external analysis.

Technical Architecture

Mobile Application
  • Developed with Flutter framework
Backend
  • Python for data processing and analysis
Database
  • MySQL for reliable data storage
Visualization
  • Matplotlib & Seaborn

Installation Guide

1
Clone the Repository

Get the source code from GitHub.

git clone https://github.com/Ad2m1109/Spendora.git
cd Spendora
2
Install Dependencies

Install the required Flutter packages.

flutter pub get
3
Run the App

Launch the application on your connected device or emulator.

flutter run
