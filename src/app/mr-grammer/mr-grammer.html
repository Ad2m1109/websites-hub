Perfect Your English with AI

Mr. Grammar is your personal AI writing assistant, designed to help you improve your English grammar, vocabulary, and overall writing skills through interactive conversations and real-time corrections.

Flutter Gemini API LanguageTool Open Source
Download Now

Key Features

Real-time Grammar Check

Get instant corrections and suggestions for your writing, ensuring accuracy and clarity.

AI-Powered Conversational Learning

Engage in interactive chats with Mr. Grammar to practice your English in a natural, supportive environment.

Track Your Progress

Monitor your improvement over time with detailed insights into your common errors and learning trends.

Text-to-Speech Integration

Listen to corrections and bot responses to improve your pronunciation and listening skills.

Multi-language Support

Start conversations in various languages to get tailored assistance (English, French, Arabic, German, Spanish).

Cross-Platform Availability

Access Mr. Grammar on your Android and iOS devices, ensuring learning on the go.

Technical Overview

Frontend
  • Flutter (Dart)
  • Responsive UI with Material Design
  • Animations for engaging user experience
Backend/APIs
  • Google Gemini API for AI conversations
  • LanguageTool API for grammar and spelling checks
  • HTTP for API communication
Data Persistence
  • shared_preferences for local conversation storage
  • Secure handling of conversation history

Installation Guide

1
Clone the Repository

Get the source code from GitHub.

git clone https://github.com/Ad2m1109/Mr-Grammer.git
cd Mr-Grammer
2
Install Dependencies

Install the required Flutter packages.

flutter pub get
3
Run the App

Launch the application on your connected device or emulator.

flutter run
