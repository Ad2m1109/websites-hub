Application Interface

The interface allows users to input housing features (e.g., Median Income, House Age, Population) and predict house values using the Gradient Boosting model. It includes visualizations such as input features vs. predicted values, median vs. predicted house value comparisons, and a distribution of predicted house values. Users can view previous predictions in a table and load them for analysis.

Installation Guide

  1. 1.
    Clone the repository: 
    git clone https://github.com/Ad2m1109/Machine-Lerning.git
cd Machine-Lerning
  2. 2.
    Create a virtual environment: 
    python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
  3. 3.
    Install dependencies: 
    pip install -r requirements.txt
  4. 4.
    Run the GUI application: 
    python gui_app.py
